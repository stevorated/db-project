# BD Project - API + CLIENT - BITCOIN-APP

## Quick Startup

```javascript

yarn compose-up --build

```

## Install modules locally

```javascript

yarn client && yarn server

```

## Start the server with nodemon(dev)

```javascript

yarn dev

```

## Start the server/client

```javascript

yarn start

```


## cleanup script for dist folder

```javascript

yarn clean

```
